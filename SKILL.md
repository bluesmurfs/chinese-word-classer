# 汉语词类划分器

给定一个语素组（可能是一个字、一个词、或一串语素），判断它是否成词，如果成词，判断它在北大版词类标准里应该归入哪一类或哪几类（兼类），并对判断依据不充分的情况明确标注出来、说明模糊的原因。

这个 skill 的核心不是"背答案"，而是把语言学判断标准转成可执行、可复现的推理过程——每一步判断都要给出测试和理由，而不是直接下结论。

## 参考文件

- `references/word_classes.md`：北大版标准里每个词类的完整定义、语法特点、原始论证。当快速判别测试给出的结果自相矛盾，或者遇到测试没覆盖到的现象时，回来查这里的完整标准。
- `references/decision_heuristics.md`：区分相邻词类的快速判别测试（捷径），以及兼类词判断的核心方法论。大多数情况下从这里就能找到答案。
- `references/calibration_examples.md`：一批已给出标准答案的例词，用来校准自己的判断方法是否与标准一致。

## 判断流程

### 第一步：成词判断

先判断这个语素组是不是一个词，而不是黏着语素或临时组合。判断标准：**词典优先，句法测试为辅**。

1. 先看这个语素组是否是规范词典（如《现代汉语词典》）或权威语料库收录的词条，是的话直接判定成词，不需要再做句法测试。
2. 如果拿不准、或者手头没有可靠的词典判断依据，退一步用句法测试辅助：这个语素组能否独立充当主语、谓语、宾语、定语等句法成分而不依赖扩展（比如加字、替换）才能使用？能独立充当，倾向于成词；只能在扩展后使用（比如"眼"要扩展成"眼睛"才能做主宾语），倾向于是不成词的语素。

注意同一个字在不同用法下成词与否可能不同（比如"眼、拳、口"作名词性成分时不成词只是语素，但作量词用时是词），这本身就是判断词类时需要留意的信息，不是矛盾。

如果判断为不成词，直接说明理由，不需要继续后面的步骤。

### 第二步：判断大类

北大标准把词分五个大类，判断词类之前先定位落在哪个大类里，能大幅缩小范围：

- **体词**（指称）：名词、量词、方位词、时间词、处所词
- **谓词**（陈述）：动词、形容词、状态词
- **饰词**（修饰，一般不能单独做主谓宾）：副词、区别词、数词、数量词、指示词
- **虚词**（不能单独充任句法成分，须依附实词）：介词、连词、语气词、助词
- **特殊词类**（按语义功能而非语法功能划分）：代词、拟声词、叹词

判断大类时优先看这个词能做哪些句法成分：能做主宾语中心语 → 体词；能做谓语补语 → 谓词；只能做修饰语或准宾语 → 饰词；不能单独做句法成分、必须依附别的成分 → 虚词；如果有称代作用或拟声/感叹功能 → 特殊词类。

### 第三步：细类判断

大类定位后，用 `references/decision_heuristics.md` 里对应的快速判别测试确定细类。这些测试是操作性的（比如"能否受'很'修饰""能否带'了/着/过+宾语'"），照着测），如果测试结果一致就可以下结论；如果测试结果互相矛盾，或者这个词根本不在任何一组对立测试的典型例词里，说明可能落在模糊地带，转到第五步。

### 第四步：兼类判断

**每判断完一个词类，都要问一遍：这个词还有没有别的常见用法/意思，对应到别的词类？** 兼类词很容易漏判，因为人容易在找到第一个说得通的词类之后就停下来。

判断是否兼类，关键标准是**词义是否发生转指**，而不是"能不能出现在别的句法位置"：

- 同一个词在不同句法位置上出现，但指的还是同一个动作/性质/事物本身，只是换了个说法——不算兼类。最常见的坑是：动词、形容词做主宾语时，很多情况下词义没变，不能因为出现在体词的位置上就判它兼名词类。
- 只有当词义真正转指为别的东西（性质 → 具有该性质的事物；性质 → 使...产生该性质的动作）时，才是兼类。

`references/decision_heuristics.md` 里"兼类词判断方法"一节给了完整的方法论和两个示范案例（危险、温暖），判断每个疑似兼类的词之前先看一遍那两个例子，把自己的推理往那个模式上对齐。

方法论：穷举这个词在语料里能想到的所有常见环境和意思，对每个环境问"这里还是原来那个意思吗，还是转指了"，逐一过一遍，而不是只测一两个环境就下结论。

**这一步最容易漏判的地方，是想不到某个不常用的义项，而不是推理错误。** 判断兼类前，如果有网络搜索工具可用，主动查一下这个词在《现代汉语词典》或"汉典 zdic.net"等工具书站点里列出的全部义项，把每个义项都过一遍第二到第四步，而不是只处理自己第一反应想到的那一两个意思；没有查证工具时，至少按 `references/decision_heuristics.md` 里"遇到拿不准的词"一节给出的检查单（文言遗留义、口语固定搭配、专业术语义、专名用法）主动想一遍，再下"只有一个词类"的结论。

### 第五步：标注模糊地带

以下情况要明确标注"这是一个模糊/存疑的判断"，并列出所有合理的可能归类及各自的理由，而不是挑一个自己觉得对的答案硬给出去：

- 快速判别测试结果互相矛盾（比如既能受"很"修饰又能带真宾语）
- 这个词的用法界于两个词类的典型特征之间，两条测试标准都只是部分满足
- 转指判断本身有争议（同一个句子里，这个词到底算没转指还是算转指，取决于怎么理解语境）
- `word_classes.md` 里对这类词的处理本身就提到了"过去的不同方案"或"学界有分歧"（比如区别词与形容词的三种分类方案）

## 输出格式

用简洁的文段说明，按下面的要点组织，不需要用表格：

1. 这个词是否成词，依据是什么（词典收录，还是句法测试，简要说明）
2. 判断的词类（如果只有一个，直接给出并说明关键测试依据；如果是兼类词，列出每个词类分别对应的用法/意思，并各自给出依据）
3. 如果存在模糊地带：明确说"这是一个存在争议/依据不充分的判断"，列出可能的归类选项，并分别说明支持每个选项的理由，让用户自己判断哪个更合适——不要在不确定的时候假装确定

## 处理多个词的输入

如果用户一次给了多个语素组，对每个词依次走完整个流程，逐个给出判断，不要因为词多就跳过某些词的兼类检查或简化判断依据。
