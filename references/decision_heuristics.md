# 快速判别测试（易混词类对）

这是用户自己总结的、用来快速区分相邻词类的操作性测试。这些是**捷径**，比逐条对照 `word_classes.md` 里的完整定义要快，但捷径在边界情况下会失灵——测试结果自相矛盾，或者词落在两个测试结果的中间，就是"模糊地带"，要停下来查 `word_classes.md` 的完整论证，而不是硬套结论。

## 动词 / 形容词 / 状态词

1. 先排除形容词：能受"很/非常"修饰 → 候选形容词（或形容词性动词兼类的形容词义）
2. 排除动词：能带"了/着/过 + 宾语" → 候选动词
3. 再排除一次动词：动词能受"不"修饰，状态词不能

结论式：
- 动词：不+/没+ 、能带真宾语、能带补语、能做非判断句谓语，且 \*（很+ 、很+～+宾都不成立）
- 状态词：能作谓语/能带"得"补语，但 \*不～、\*没～、\*很～、\*～宾、\*～补 都不成立
- 区别词（放在这组一起对比更清楚）：\*很～、\*不～、不能单独做谓语、\*～补、\*～准宾语

**做主宾语不改变动词/形容词本身的词性**——用作主宾语的动词、形容词仍看作动词、形容词，不要因为出现在主宾语位置就认为兼了名词类。这是最容易犯的错误之一。

## 形容词 / 状态词 / 区别词

1. 排除形容词：能否受"很/非常"修饰
2. 判断做定语时能否加"的"、能否作谓语：能 → 状态词；不能 → 区别词

## 位置词（方位词/名词/副词/时间词/处所词）

- 方位词：常能搭配"往+"测出表方向的用法（往前、往上）；表相对位置，需要参照，能后附于参照性名词构成方位结构（桌子上、长江一带）
- 时间词：能做主语（试"～有课"）、能放在"在"之后、能用来回答"什么时候"提问 → 时间词；不能 → 副词
- 处所词：能否直接放在"在/到/往"后面且不需要再加方位词；处所词后面不能再加方位词（不能说"在当地里"），这是和方位词、普通名词的重要区别
- 表示相对位置 → 方位词；表示绝对位置 → 处所词

## 量词 / 名词

能单独做主宾语 → 名词；不能 → 量词。例如"一/m方/l""本/h章/l"这种最小对立。

另一条交叉验证：量词本身不能再受数量词组修饰，如果某个量词候选词能被"数词+量词"再套一层（比如"一个小时"里的"小时"能被"一个"修饰），说明它其实是名词兼量词，不是纯量词——很多看起来像量词的词都是这种情况，遇到"数+X"结构里的X要顺手测一下。

## 数量词 / 名词

能单独做主宾语，且前面还能再受"数词+量词"结构修饰（如"一个大量"不成立，但"一个部分"可以）→ 名词；不能受数量结构再修饰 → 数量词。

不少数量词还兼副词，测试能否直接修饰谓词性成分来确认（大量生产、少量存在、部分属实）。

## 副词

能且只能修饰动词作状语——这是副词最本质的鉴别特征，凡是发现某词只能做状语、不能做其他任何句法成分，基本可以定为副词。

反过来也可以测：副词本身一般不能被"不/很"修饰（\*不简直、\*很难道），如果一个词能被"不/很"修饰，大概率不是纯副词，而是形容词/动词的候选，要回去重新走一遍谓词的判断流程。

## 数词

就是数字本身（含系数词、位数词及其组合），核心测试是能否修饰量词构成数量结构。

## 指示词 / 指示代词

前者只能做定语（不能单独做主宾语），且能进入"+数量"结构（某一个、每一个、这一个）；后者相当于名词，可以单独做主宾语。同一个字（如"这、那"）在不同位置可能分属两类，不是兼类，是**位置决定的功能差异**，参见 `word_classes.md` 代词一节。

## 介词 / 连词

前者是引出对象，后者只是连接（相当于"and"）。

介词的结构标志：后面接名词性成分构成介词结构，这个介词结构整体还要再接一个谓词性成分做修饰（对他说、把书放下、朝前走去）——如果"介词+名词"本身就独立做了谓语、后面没有别的谓词性成分，那就不是介词而是动词。

连词的结构标志：连接的是结构地位平等的两部分（词、词组、分句或句子），比如"因为下雨，所以取消了"里"因为"接的是分句而不是名词。

用插入变换、省略变换验证介词；复数变换、拆分变换验证连词——**满足一个方向的任一条即可判定该方向成立**，两个方向都满足就是兼类。
"因/因为"接名词则为介词，接谓词则为连词。（完整的变换测试例句见 `word_classes.md` 连词一节）

## 语气词 / 叹词 / 拟声词

1. 语气词：放在句尾（或句中停顿前）
2. 判断是否独立成句：独立成句 → 叹词；不独立成句 → 拟声词
3. 很多时候这三类是"有一个必有其他两个"，即同一形式兼属三类中的不止一类，需要多找语境判断具体是哪种用法在起作用，而不是选一个了事

## 关系类词的动词 / 形容词判断（相同、相反、相似、相等、相符一类）

这类表示两事物之间关系的词，用常规的"能否受'很'修饰"测试经常测不准，因为很多这类词受"很"修饰时本身就很勉强，不能简单地"测不出很+就判定不是形容词"。更准确的测试：

- "很~"和"很不~"都能说得通 → 形容词（如"相同"：很相同虽然也不算特别自然，但整体上作为性质形容词处理）
- "很~""很不~"都不成立，但"不~"本身成立 → 动词（如"相反"：很相反、很不相反都不成立，但"不相反"成立，按关系动词处理，跟`word_classes.md`动词一节列的"等于"是同一类）

遇到"相X"这一类词，先把这两个测试都过一遍再下结论，不要只测"很~"一项就停。

## 程度/维度形容词的名词兼类（长、宽、高、厚、深、粗一类）

"高、长、宽、厚、深、粗"这些表可测量维度的性质形容词，除了常规的性质形容词用法（高山、很高、这座山很高）外，在数学、几何、测量语境下常常还有一个名词用法，指这个维度本身、作为一个可以被赋值的量，比如"这个正方体的高是2米""长3宽2高5"。

测试方法：能否做主语，后面接"是+数量结构"作谓语（"高是2米"），或者受"的"字定语修饰、整体做主宾语（"正方体的高"）。满足就是名词用法，和常规的性质形容词用法（做谓语、受"很"修饰）是两个不同的词类，判定兼类。这个模式比"危险"那种转指更规整、更容易预判——遇到维度类形容词时，主动测一下这个数学语境下的名词用法，不要因为"高"最常见的用法是形容词就只测这一种。

## 遇到拿不准的词，主动去查证义项，而不是只凭脑子里最常见的用法判断

这是这个 skill 最大的失败模式来源：语言模型（包括执行这个 skill 的你）在判断一个词的时候，容易只想到这个词最高频、最直觉的一两个意思，然后就直接下结论说"只有这一个词类"或者"不构成兼类"。但很多兼类恰恰藏在不那么高频的义项里——文言遗留的用法（比如"每"表"常常"）、口语化的固定搭配（比如"白了他一眼"里的"白"）、专业术语语境下的用法（比如"高"在几何里做名词）、方言或半自由语素的用法（比如"拳着腿"）。

如果拿到工具（比如网络搜索），在判断一个词、尤其是要断言"没有别的词类"之前，主动查一下这个词在规范词典（如《现代汉语词典》）或"汉典 zdic.net"这类工具书站点里列出的**全部义项**，把每一个义项都过一遍决策流程，而不是只处理自己第一反应想到的那一两个。如果没有查证工具，至少要在心里明确走一遍这个检查单：这个词有没有文言/书面语遗留义？有没有方言/口语固定搭配？有没有专业术语（数学、法律、医学等）语境下的特殊用法？有没有作为姓氏/地名的专名用法（这类一般不计入普通词的兼类，但要留意区分）？

如果查完还是拿不准某个义项是否存在、是否够格算独立词类，按第五步的要求把这个不确定性写出来，而不是悄悄略过。

## 助词

被剩下的虚词——即不是介词、连词、语气词的虚词，归入助词。因为是剩余类，成员共性弱，个性强，遇到助词候选词时优先去 `word_classes.md` 查具体是哪个助词及其功能，不要试图用统一标准套。

## 位置对词类的可能性的限制（可用于交叉验证）

- （指示词+）数词+量词+名词
- 带"的"定语前可以出现数量词（但不能出现区别词，这是数量词/区别词最重要的鉴别点）
- "在"+表示空间位置的成分（（名词+）方位词/处所词）
- 能进入"非常/很+～+宾语"位置的动词都是心理动词
- "动词+～+了"之间能填入的补语，一般由不及物动词、形容词充任
- 做定语的成分一般是形容词、区别词、名词

## 兼类词判断方法（用户的核心方法论，最重要的一条原则）

兼类词的本质：一个同形的词有不同的词义，不同词义归属不同词类，但**一个词义终究只能划到一个词类**。判断是否兼类，关键不是"这个词能不能出现在两种句法位置"，而是"出现在不同句法位置时词义有没有发生转指"：

- 如果词义没变，只是句法位置变了（例如动词、形容词做主宾语时依然指称这个动作/性质本身），**不算兼类**——这是最容易过度判断兼类的地方，要主动排除。
- 如果词义发生了转指（不再指这个动作/性质/属性本身，而是转指"具有该性质的事物"或由该动作引出的事物），才是兼类。

**判断步骤**：找全这个词在语料里出现的所有环境和所有意思，对每个环境逐一提问"这里的意思是不是还是原来那个动作/性质，只是换了个位置说" —— 如果是，不算新词类；如果发生了转指，才记一个新词类。

**用户给出的两个示范案例**（可直接作为校准参照）：

1. **危险**：形容词（可被"很"修饰："很危险"）。做主语时（"危险来自于内部"）指"危险的事情"，不再指"危险"这个性质本身 → 发生转指 → **危险 = 形容词 + 名词兼类**。
2. **温暖**：形容词。做主语时（"温暖是这个季节的主题"）仍然指"温暖"这个性质本身，没有转指 → **不算名词兼类**。但做谓语时（"这句话温暖了我"）表示"使人变得温暖"这个动作，发生了转指（性质→致使动作）→ **温暖 = 形容词 + 动词兼类**。

遇到形容词/动词做主宾语的情况，套用这个转指测试，而不是简单地看"能不能做主宾语"。
